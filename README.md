# EIS: Software module to import EIS data from text files 
fl\_eis\_tdim is a software to extract electrochemical impedance data from the hardware text output.
Documentation will be provided separately for all EIS suit. 

fl\_eis\_tdim requires relatively fresh C++ compiler. 
Library can be built using **CodeBlocks IDE**  and **MinGW** compiler. fl\_eis\_tdim requires:
 **fmtlib** <https://github.com/fmtlib>. 
 **libconfig 1.7.3** from here: https://github.com/nenin-sc/libconfig1.73alt Priscine libconfig will require minor code corretion.
 **FLTK Generic Form Context** <https://github.com/nenin-sc/FLTK-Generic-Form-Context>
 **fltk_plotXY3** <https://github.com/nenin-sc/fltk_plotXY3>
 **GSL 2.8** <https://www.gnu.org/software/gsl/>. 
 **pystring** https://github.com/imageworks/pystring 

It was not tested out of Windows, so potentially some issues may occure.
